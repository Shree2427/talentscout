# TalentScout Hiring Assistant 🤖

## Overview
This application is an AI-powered hiring assistant designed to perform initial candidate screening.

## Model Used
An open-source pre-trained LLM (DialoGPT-medium) was used to ensure:
- Cost efficiency
- Scalability
- No dependency on paid APIs
- Full offline capability

## Tech Stack
- Python
- Streamlit
- HuggingFace Transformers
- PyTorch

## Features
- Conversational screening
- Context retention
- Dynamic response generation
- Local LLM inference

## How to Run

pip install -r requirements.txt
streamlit run app.py
